Week 6 (11/03 - 15/03): CI / CD omgeving voor de backend gereed en twee nieuwe functionaliteiten in de app

De week

De eerste dag van deze nieuwe week ben ik begonnen met het ontwikkelen van een nieuwe functionaliteit in de app, namelijk het aanvinken van een favoriet door de gebruiker. Als de gebruiker op een hartje klikt bij een artiest wordt deze toegevoegd aan zijn of haar favorieten. Vervolgens heb ik nog gewerkt aan de de front-end van het rooster welke ik af heb gemaakt. Door het gebruik van timeboxen had ik aan het eind van de dag ook nog tijd vrij gemaakt voor het opzetten van de CI / CD testomgeving voor de backend welke ik ook af heb gekregen.

Dinsdag heb ik nog een aantal kleine bugs uit het rooster en favorieten deel van de app gehaald. Hierna kon ik vervolgens beginnen aan de derde functionaliteit van de app waarin de gebruiker de meest geschikte bar / wc kunnen vinden. Hier heb ik vandaag de backend voor geschreven. Daarna ben ik ook nog bezig geweest met het maken van een dockerfile en integratie met jenkins zodat deze automatisch op een docker omgeving wordt gezet wanneer er naar develop is gepushed.

Op de woensdag heb ik nog een klein aantal foutjes gehaald uit de backend van de derde functionaliteit waarna deze doorgezet kon worden. Vervolgens heb ik gewerkt aan de front-end van deze functionaliteit zodat alle plekken getoond kunnen worden in de juiste volgorde. Ook heb ik vandaag twee gesprekken gehad, 1 gesprek met Bart over Paaspop en hoe dat ik zijn werk zal gaan en het tweede gesprek was met Henk Brands (business unit manager) en was mijn eerste feedback gesprek. Henk was erg positief. Aan het eind van de dag heb ik het voor elkaar gekregen om via jenkins de docker container te draaien op de vm.

Donderdag heb ik de derde functionaliteit (meest geschikte wc / bar) afgemaakt op zowel front-end gebied als back-end. Deze heb ik op develop gezet en vervolgen ben ik bezig geweest met een fout uit te zoeken in het docker file van de back-end.

Vrijdag heb ik de gehele dag gewerkt aan de CI / CD omgeving en de database zodat de configuratie goed staat zodat wanneer de app op de playstore staat, deze gebruikt kan worden met een externe database. Dit is uiteindelijk allemaal gelukt en ben ik begonnen met het opzetten van de app in de playstore. Hier ben ik zondag ook nog mee bezig geweest en de eerste versie staat nu als beta op de playstore. Vrijdag had ik ook nog het voortgangsgesprek met Bart, maar zoals altijd ging dit erg goed.

Problemen en feedback

Deze week heb ik niet erg veel problemen ondervonden en ook heb ik geen feedback ontvangen.
