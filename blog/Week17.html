Week 17 (27/05 - 29/05): Definitieve documenten, presentatie teksten en begin aan testen van React Native

De week

Deze week zal ik in één korte paragraaf beschrijven aangezien ik drie dagen heb gewerkt en vrijveel aan dezelfde taken. Op maandag ben ik de hele dag nog bezig geweest met het definitief maken en controleren op grammaticale en spellings fouten van alle documenten in mijn portfolio. Vervolgens heb ik de dinsdag besteed aan het schrijven van de eerste versie van de presentatie tekst voor de afstudeerzitting. Ook heb ik de tekst geschreven voor tijdens de demo van de applicatie. Woensdag heb ik in de ochtend mijn back-end code gedocumenteerd voor betere onderhoudbaarheid en ben ik begonnen aan het schrijven van React Native tests. Hiervoor heb ik gekeken welk framework ik hiervoor kon gebruiken en hoe dat werkte / welke soort tests mogelijk waren. Hiermee ga ik maandag weer verder.
